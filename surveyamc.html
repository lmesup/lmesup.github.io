SurveyAMC

The SurveyAMC package is targeted to create machine-readable questionnaires for selfadministered paper surveys in high typesetting quality. The package provides a LaTeX style for designing the questionnaire’s general layout, and for formatting and placing the questions, the answers, the check-boxes, the completion instructions and the filter instructions. After the completion of the survey, the respondents’ answers can be automatically processed into a fully structured file using the freely available software Auto-Multiple-Choice (Version 1.4.0) by Alexis Bienvenüe. In addition to the LaTeX style, the package contains companion programs for a more general system in which both, paper and Online questionnaires, as well as several by-products are automatically created from the same source file.

Notes on use

  • To create machine-readable questionnaires with SurveyAMC you will need a latex distribution, an editor, a pdf-viewer and the LateX package "surveyamc.sty". Surveyamc.sty is made available on the SurveyAMC project repository. You can download it there, copy it to your latex working directory and then load it with \usepackage{surveyamc} in the preambel of your tex document. For more detailed information see the documentation.
  • For processing the completed questionnaires into a dataset you will need the Auto-Multiple-Choice Software (Version 1.4.0). Informations about installation and usage you can find at www.auto-multiple-choice.net.

Documentation

The SurveyAMC package is fully documented. The documentation helps you to get started with your own survey project. It contains:

  • installation guidelines
  • example questionnaire
  • code snippets for a variety of question format options
  • style template for the general page layout
  • description of the "DRY" workflow for generating several survey products from one source file
The full documentation of SurveyAMC ist hosted on the project repository.

Questionnaire Templates

Different templates may help you to create your first questionnaire with SurveyAMC. You can find all templates on the SurveyAMC project repository. The list of templates is constantly being updated. Follow the link, copy the files to your project directory and adjust them to fit your personal needs.
